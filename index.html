How Roblox Lookup Works

Type a game name, universe ID, or place ID to discover experiences. For users, enter a username or user ID. The tool fetches publicly available metadata (name, creator, visits, like ratio, presence and counts) and lets you sort, filter, bookmark, and share deep links. No login is required.

Tips

  • Use G/U on your keyboard to switch modes.
  • Sort by Visits, Playing, Favorites, Likes % or Name. Apply filters to hide low-traffic games.
  • Bookmark favorites and Export them as a JSON file you can import on another device.
  • “🎲 Surprise me” will pick a random result from your last search.
